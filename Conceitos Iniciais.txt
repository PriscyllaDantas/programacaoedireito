Os formatos de arquivos de dados podem ser divididos em Proprietários e Abertos. Sendo exemplos de Proprietários: Microsoft Excel (.xls, .xlsx); SAS (.sas7bdat); Oracle Database File (.dbf). E exemplos de Abertos: Comma-Separated Values (.csv); JavaScript Object Notation (.json); HDF5 (.h5).
Quando os dados públicos são disponibilizados em formatos proprietários, podem haver barreiras de acesso para pessoas ou organizações que não possuem o software ou licenças necessárias para abrir e manipular esses arquivos. Enquanto os dados públicos disponibilizados em formatos abertos, são acessíveis a um público mais amplo, pois não requerem software específico para visualização ou processamento. Ao se utilizar formatos de arquivos de dados abertos para dados públicos, promove-se uma maior democratização e utilização eficaz dos dados públicos, isto é, os dados públicos se tornam mais acessíveis, interoperáveis e transparentes.
